# Assignment-3 Part-A — Project Proposal

## Submission Requirements

- **Who submits:** Every student individually
- **File naming:** `Assignment-3-PartA-<StudentName>-<StudentID>.doc/pdf`
- **Example:** `Assignment-3-PartA-NabinSharma-12345678.pdf`

## What to Include

Your Part-A submission should be a project proposal covering:

1. **Project Title**
2. **Team Members** (Name and Student ID for each member)
3. **Project Abstract / Summary** — brief description of the problem you are solving
4. **Dataset** — what dataset(s) you plan to use
5. **Proposed Method / Model** — what deep learning approach you intend to use (e.g., CNN, YOLO, transfer learning, etc.)
6. **Expected Outcomes** — what results you expect to achieve
7. **Timeline / Plan** — how you plan to divide the work across the intermediate deadlines

## Notes

- All team members must submit Part-A individually.
- The document content can be the same for all team members.
- If you have not formed a group yet, you can still submit your individual project idea. The teaching team will assist in forming groups after Part-A submissions.
- Group size: **minimum and maximum of 3 students**.
